      Tema básico para a nova versão do Portal Padrão IDG 2.0

      Novo tema para a versão 2.0 da Identidade Digital do Governo Federal. O novo tema é muito mais simples e fácil de customizar
      por Governo do Brasil — última modificação 20/11/2017 15h29 — Histórico

      Este é um tema muito simples e enxuto para o Plone para ser utilizado nas instalações padrão da Identidade Digital do Governo Federal.

      HTML

      Ao invez de utilizar diversos divs o tema utiliza várias tags padrão do HTML 5

      Styles

      Todos os estilos são definidos em um arquivo SCSS único, comum para todos os temas existentes. Cada tema possui apenas um simples arquivo de configuração de váriaveis, que altera toda a aparencia do portal.

      Grid

      Este tema utiliza FLEX para a criação de grids extremamente simples em lugares como cabeçalho, footer e nas colunas principais.

      Alem disso este tema implementa os estilos grid básico do Bootstrap 3.0, permitindo a utilização do portal com 12 colunas ao invez do tradicional grid de 16 colunas do Plone
